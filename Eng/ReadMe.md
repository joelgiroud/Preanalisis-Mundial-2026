This data repository explores the FIFA World Cup 2026, the first tournament to be 

co-hosted by three neighboring countries: \*\*USA, Canada, and Mexico\*\*.



\---



\## Part 1: R Project

\*\*Data engineering + Monte Carlo simulation\*\*



Implements data validation, cleaning, merging, and runs \*\*10,000 tournament simulations\*\* 

using FIFA rankings. Generates all processed CSVs consumed by Power BI.



\*\*Recommended for:\*\* Data engineers, R enthusiasts, simulation modeling



\---



\## Part 2: Power BI Dashboard

\*\*Interactive analytics + HTML custom design\*\*



5-page dashboard visualizing venues, rankings, historical stats, and simulation results 

with custom HTML Content styling (dark theme, responsive cards, conditional colors).



\*\*Recommended for:\*\* BI analysts, dashboard design, data visualization
