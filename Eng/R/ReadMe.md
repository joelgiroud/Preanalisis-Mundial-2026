# 🏆 Simulating FIFA World Cup 2026
### Data Analysis Portfolio | R · ggplot2 · Monte Carlo

---

## 📋 Overview

Data analysis project answering 3 key questions about FIFA World Cup 2026
through exploratory analysis, predictive modeling, and visualization:

1. How are the 104 matches distributed across the 16 venues?
2. Which national team has the highest probability of winning?
3. How much does local hosting impact Mexico's chances?

---

## 🗂️ Project Structure


```
mundial_2026/
├── data/ # Processed data (generated by script)
├── output/ # Exported charts and reports
├── main.Rmd # Main script (fully reproducible)
└── README.md # This file
```

## 🔬 Methodology

- **Data validation:** `pointblank` (completeness, validity, uniqueness)
- **Predictive model:** Logistic function over FIFA points difference
- **Simulation:** Monte Carlo · 10,000 tournaments · `set.seed(2026)`
- **Goals distribution:** Poisson with mean proportional to team strength
